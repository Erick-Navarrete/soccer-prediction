
Windows Task Scheduler Setup Instructions:

1. Open Task Scheduler:
   - Press Win+R, type 'taskschd.msc', and press Enter

2. Import the task:
   - Right-click on 'Task Scheduler Library'
   - Select 'Import Task...'
   - Browse to: D:\Project_App\soccer-prediction\outputs\tableau_data\tableau_refresh_task.xml
   - Click Open

3. Or create manually:
   - Click 'Create Basic Task'
   - Name: 'Tableau Data Refresh'
   - Trigger: Daily
   - Start: 6:00 AM
   - Action: Start a program
   - Program/script: D:\Project_App\soccer-prediction\refresh_tableau_data.bat
   - Start in: D:\Project_App\soccer-prediction

4. Test the task:
   - Right-click on 'Tableau Data Refresh'
   - Select 'Run'

5. Monitor:
   - Check logs in: D:\Project_App\soccer-prediction\outputs\tableau_data/refresh.log
